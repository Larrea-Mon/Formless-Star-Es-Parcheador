INSTRUCCIONES DE JUEGO
Formless Star - creado por samanthuel louise gillson (splendidland)


¡Hola, te damos la bienvenida a Formless Star!

En este juego, viajas a un planeta cambiante para recolectar datos de animales.

¡Cada vez que sales de la nave, el mundo se genera aleatoriamente!

Puedes destruir los obstáculos en tu camino y cruzar el mar colocando puentes.

¡Hay 60 animales distintos! Cada vez que encuentres uno, se guardará tu progreso.


CONTROLES
Flechas - moverse, controlar menú
Espacio/Intro/Ctrl - usar, confirmar, destruir obstáculos/colocar puentes
Esc/Alt - abrir y cerrar el menú principal
